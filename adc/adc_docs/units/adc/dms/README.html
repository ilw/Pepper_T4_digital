class pepper_adc.PepperDataConverter_V1(vdd: float = 1.8)[source]#

Bases: object

Models the Pepper capacitor CDAC array with MES2+DWA functionality.

This is a single-ended representation of the fully-differential capacitive dac used in the Pepper NS ADC. Metrics will see 6dB improvement.

The design does not use split capacitors and instead segments the array into 4 sections: one DWA MSBs, two MES2 LSBs, and coupling to the analogue quantization noise-shaping filter.

The MSB section uses two 832 fF unit capacitors for feeding back the first 4 bits. The same 832 fF capacitor is used to couple the analogue loopfilter during sampling. This is done so that the common-mode reference voltage is used as little as possible and kept quiet.

The LSB section uses two 6 bit CDACs with a ping-pong configuration. Here the unit capacitor is 51.4 fF.

The total sampling capacitance is 15x 2x 832 fF or around 25 pF on both positive and negative analogue inputs.
